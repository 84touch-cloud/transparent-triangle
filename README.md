# 透明な三角形

**透明な三角形** は、空間知覚をテーマにしたミニゲームです。  
画面上には三つの点がありますが、実際の三角形は見えません。

プレイヤーは、色付きの点を動かしながら  
「そこにあるはずの三角形」を想像して再現します。

---

## 🎮 遊び方

1. 上のキャンバスに、正解となる三角形の頂点が配置されています  
   （※ 判定までは見えません）
2. 下のキャンバスで、  
   青・緑・赤の点をタップ（クリック）して配置します
3. 「決定」を押すと判定されます

---

## 🏆 判定

- **Perfect**  
  ほぼ完全に一致しています
- **Good**  
  かなり近い位置です
- **Miss**  
  惜しい！もう一度挑戦できます

スコアは、各頂点の距離の合計で計算されます。
